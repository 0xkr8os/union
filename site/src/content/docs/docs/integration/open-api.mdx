## REST

A REST interface for state queries.
- Obtain information on open IBC channels and connections,
- Query the Union chain for the latest block, transaction, validator set,
- Encode, decode, simulate and broadcast transactions.

## JSON-RPC

A JSON-RPC interface for obtaining information about the network.
- Obtain informations about the node APIs,
- Query transactions and broadcast APIs,
- Run `ABCI` queries.
